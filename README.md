### Building

Requirements:

  * Meson 0.49 or newer
  * Ninja 1.8.2 or newer
  * A C compiler
  * A C++ compiler

#### To build from command line, go into one of the project directories, then:

To initialize the builder:

    meson build

To build the project from then on:

    ninja -C build
